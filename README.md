# Fake News Detection using Machine Learning

## Overview
A Machine Learning project that classifies news articles as *Fake* or *Real* using Python and Natural Language Processing (NLP).

## Features
- Fake and Real news classification
- Data preprocessing and text cleaning
- Machine Learning model training
- News prediction
- Built using Google Colab

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Dataset
- True.csv
- Fake.csv
